# `json_to_map`

Restituisce tutte le chiavi di una mappa come un array.

## Sintassi

json_to_map(_json string_)

## Argomenti

* _json string_ una stringa json

## Esempi

* ``json_to_map('{"a":12,"b":["rre","rra"]}') â†’ <map: a:12,<array: 'rre','rra'>>``
