# `translate`

Restituisce una versione traslata di una geometria. I calcoli sono effettuati nel sistema di riferimento spaziale di tale geometria.

## Sintassi

translate(_geometry, dx, dy_)

## Argomenti

* _geometry_ una geometria
* _dx_ delta x
* _dy_ delta y
